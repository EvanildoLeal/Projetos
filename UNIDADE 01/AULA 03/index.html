As listas não ordenadas são aquelas que não possuem uma ordem sequencial, ou seja, não possuem uma numeração ou uma letra que indica a posição do item na lista.

Exemplos de listas não ordenadas:

  • MYSQL
  • SQL Server
  • PostgreSQL
  • Oracle
  • SQL Lite
